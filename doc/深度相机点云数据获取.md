1. 设置/opt/ros2/cyberdog/share/realsense2_camera/launch/realsense_align_node.launch.py中的allow_no_texture_points参数为true

2. 启动以下节点与服务

```
ros2 lifecycle set /cyberdog_2/camera/camera_align configure
ros2 lifecycle set /cyberdog_2/camera/camera_align activate
ros2 service call /cyberdog_2/camera/realsense_frame_service std_srvs/srv/SetBool "{data: true}"
```

话题/cyberdog_2/camera/depth/color/points
