# 网线连接
通过网线连接机器人的网口，并将用户电脑IP设为：192.168.44.100/255.255.255.0
```bash
ping 192.168.44.100     #用户电脑被分配的IP
ssh mi@192.168.44.1     #登录NX应用板，密码123
ssh root@192.168.44.233 #可从用户电脑直接登录运控板
```

# cyberdog WIFI设置


```bash
sudo iwlist wlan0 scan | grep -i "ESSID"  # 搜索wifi
nmcli device status # 查看连接状态
```


连接wifi，后续启动自动连接路由器wifi
```bash
sudo nmcli device wifi connect "wifi名称" password "密码" ifname wlan0
# 例如
sudo nmcli device wifi connect "drgk_5G" password "drgk-13308080547" ifname wlan0
```

# 多机通信
本机使用ros2环境获取cyberdog话题，可使用rviz2进行可视化

## cyberdog端修改
**修改/etc/mi/cyclonedds.xml文件**

核心改动NetworkInterfaceAddress和AllowMulticast字段内容
```bash
<?xml version="1.0" encoding="UTF-8" ?>
<CycloneDDS xmlns="https://cdds.io/config" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"    xsi:schemaLocation="https://cdds.io/config https://raw.githubusercontent.com/eclipse-cyclonedds/cyclonedds/master/etc/cyclonedds.xsd">
    <Domain id="42">
        <General>
            <NetworkInterfaceAddress>wlan0</NetworkInterfaceAddress>
            <AllowMulticast>true</AllowMulticast>
        </General>
        <Discovery>
            <ParticipantIndex>auto</ParticipantIndex>
            <MaxAutoParticipantIndex>256</MaxAutoParticipantIndex>
            <Peers>
                <Peer address="localhost"/>
            </Peers>
        </Discovery>
    </Domain>
</CycloneDDS>

```

**修改cyberdog命名空间**

修改下面两个文件get_namespace()函数的返回值，修改为"cyberdog_1"、"cyberdog_2"等等
```
/opt/ros2/cyberdog/share/cyberdog_bringup/bringup/manual.py

/opt/ros2/cyberdog/lib/python3.6/site-packages/mi/cyberdog_bringup/manual.py
```

## 本机ros2环境配置

后续每次进本机的docker环境时，均需使用以下指令配置ros2环境，才可接收到cyberdog2机器上的话题，或设置一个脚本，让docker启动时自动执行以下命令
```bash
### ROS2 ###
source /opt/ros/galactic/setup.bash
export RMW_IMPLEMENTATION=rmw_cyclonedds_cpp
export CYCLONEDDS_URI=file:///home/lee/.cyclonedds.xml		# 更改为实际的路径
export ROS_DOMAIN_ID=42
```


在本机的合适位置创建.cyclonedds.xml文件，内容如下，NetworkInterfaceAddress设置为本机的wifi端口
```bash
<?xml version="1.0" encoding="UTF-8" ?>
<CycloneDDS xmlns="https://cdds.io/config" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"	xsi:schemaLocation="https://cdds.io/config https://raw.githubusercontent.com/eclipse-cyclonedds/cyclonedds/master/etc/cyclonedds.xsd">
	<Domain id="42">
		<General>
			<NetworkInterfaceAddress>wlp0s20f3</NetworkInterfaceAddress>
			<AllowMulticast>true</AllowMulticast>
		</General>
		<Discovery>
			<ParticipantIndex>auto</ParticipantIndex>
			<MaxAutoParticipantIndex>256</MaxAutoParticipantIndex>
 			<Peers>
				<Peer address="localhost"/>
			</Peers>
 		</Discovery>

	</Domain>
</CycloneDDS>

```

## PC端如果收不到机器人上的话题？

重启ros2 的守护进程
```bash
ros2 daemon stop
ros2 daemon start
```
